Assalamualaikum, Wr. Wb. Perkenalkan nama saya MHD. SYAHRIZAL biasa dipanggil Syahrizal atau Izal. Kesibukan saya saat ini yaitu sedang aktif berkuliah pada jurusan Teknologi Informasi di Universitas Sumatera Utara. Pada saat ini saya sedang mengikuti program MSIB (Magang Studi Independen Bersertifikat) di NF Computer dengan subjek Fullstack Web Development. Saya tinggal di daerah pesisir pantai yakni di Kecamatan Pantai Cermin. Kawasan saya tinggal merupakan sebuah daerah wisata bahari yang paling terkenal di Kota Medan. Di daerah ini saya tumbuh dan berkembang mulai dari kecil hingga menempuh pendidikan tingkat atas. Dari daerah tersebutlah saya belajar banyak hal yang saya pergunakan untuk bertahan di bangku perkuliahan. Pada bangku perkuliahan saya merantau ke Kota Medan, di kota tersebut banyak sekali hal-hal baru yang dapat temui karena daerah perkotaan memiliki social activity yang sedikit berbeda dengan daerah perkampungan. Akan tetapi, hal tersebut hanya berlangsung 1 semester saja, dan kemudian pandemi covid-19 melanda dan membuat saya harus kembali ke kampung halam saya dan berkuliah dari rumah (Study From Home).

Walaupun berkuliah dari rumah, hal tersebut lantas tidak menghambat semangat saya untuk terus menimbah ilmu pengetahuan. Saya pribadi merasa beruntung karena berada dalam jurusan Teknologi Informasi karena dengan jurusan tersebut saya dapat dengan mudah untuk beradaptasi dengan perubahan yang ada, khususnya perubahan yang melibatkan teknologi di dalamnya. Tidak hanya itu saja, dengan masuknya saya kedalam jurusan teknologi informasi ternyata banyak hal-hal baru dan unexpected yang baru saya temukan. Dengan hadirnya hal-hal baru tersebut lantas membuat pikiran saya senantiasa melahirkan ide-ide yang baru pula.

Oleh karena itu, demi mewujudkan ide-ide tersebut maka saya mengambil kesempatan dan tantangan untuk mengikuti program MSIB di NF Computer agar dapat menguasai bidang Fullstack Web Developer. Saya berharap nantinya dapat mewujudkan ide saya dan semoga cerita saya ini dapat dijadikan sebagai sebuah inspirasi bagi kalian yang sedang mengunjungi website saya ini. Apabila dalam penulisan dan penyampaian informasi terdapat beberapa kesalahan dan kekurangan saya mohon maaf, kepada Allah SWT. saya mohon ampun. SELAMAT DATANG DI WEBSITE SAYA!
